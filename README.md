# KMS

A simple key management system to store user encrypted key.

### Tech Stack
- Node JS
- React JS
- MongoDb (Mongo Atlas)

### Quick Start
- open two terminals
- First Terminal:
    
        cd server
        npm install
        npm run server
      
- Second terminal
    
        cd client
        npm install
        npm run start
        
### API

This branch implements google-oauth using nodejs.

Api for google authentication: `localhost:5000/auth/google`
